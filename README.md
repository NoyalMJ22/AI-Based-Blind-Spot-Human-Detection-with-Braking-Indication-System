# 🚗 AI-Based Blind Spot Human Detection with Braking Indication System

## 📌 Overview

This project is an AI-based system that detects humans in a vehicle’s blind spot and provides braking indication alerts to improve road safety.

---

## 🎯 Features

* Real-time human detection
* Blind spot monitoring
* Braking alert indication
* Detection logging system

---

## 🧠 Technologies Used

* Python
* OpenCV
* YOLOv8
* NumPy

---

## ⚙️ Installation

```bash
git clone https://github.com/NoyalMJ22/AI-Based-Blind-Spot-Human-Detection-with-Braking-Indication-System.git
cd AI-Based-Blind-Spot-Human-Detection-with-Braking-Indication-System
pip install -r requirements.txt
```

---

## ▶️ Usage

```bash
python main.py
```

---

## 📊 Output

* Logs are saved in CSV and JSON format
* Detection results stored in the logs folder
